Axolotl is a free and open-source framework for post-training and fine-tuning large language models. Whether you're doing instruction tuning with SFT, preference alignment with DPO, or reinforcement learning with GRPO, Axolotl unifies the full pipeline — dataset preprocessing, training, evaluation, and inference — into a single YAML configuration file.

## Get started

<Steps>
  <Step title="Install Axolotl">
    Install via pip or uv with your preferred CUDA version:
    ```bash
    uv pip install --no-build-isolation axolotl[flash-attn,deepspeed]
    ```
  </Step>
  <Step title="Fetch example configs">
    Download ready-to-use example configurations:
    ```bash
    axolotl fetch examples
    ```
  </Step>
  <Step title="Train your first model">
    Run a LoRA fine-tune on a 1B Llama model — works on most GPUs:
    ```bash
    axolotl train examples/llama-3/lora-1b.yml
    ```
  </Step>
  <Step title="Run inference">
    Test your trained model interactively:
    ```bash
    axolotl inference examples/llama-3/lora-1b.yml --lora-model-dir="./outputs/lora-out"
    ```
  </Step>
</Steps>

<Note>
  Axolotl requires an NVIDIA GPU (Ampere or newer for bf16 and Flash Attention) or an AMD GPU, Python ≥3.11, and PyTorch ≥2.9.0.
</Note>
